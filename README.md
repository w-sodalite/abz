# abz

批量将`zip`和`tar.gz`相互转换的小工具。

> 实验性支持`7z`压缩包

## 安装

> 目前由于依赖`iced`的`dev`版本，没有发布到`crates.io`，后续该`dev`版本发布正式版本后会同步发布到`crates.io`中。

克隆仓库到本地

```shell
git clone https://github.com/w-sodalite/abz.git
```

编译安装

> 需要先安装rust环境

```shell
cargo build --release
```
